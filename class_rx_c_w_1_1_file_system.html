static AsyncFile* RxCW::FileSystem::open ( const std::string &  path,
const std::string &  mode 
)
static

Open the given file with the specified modes.

Parameters
pathThe file to open.
modeThe mode to open the file with.
Returns
AsyncFile* The resulting object to manipulate the file with. Can be:
  • r: Open the file for reading, does not create the file if it does not already exists.
  • w: Create a file for writing, truncate the file if it already exists, create it otherwise.
  • a: Append to a file, create it if it does not already exists.
  • r+: Open a file for read/write, does not create the file if it does not already exists.
  • w+: Create a file for read/write, truncate the file if it already exists, create it otherwise.
  • a+: Open a file for read/write, writings will be appended to the file. Create the file if it does not already exists.
